## **QUICK START GUIDE**

### **Prerequisites**

```bash
# System Requirements
- Node.js >= 18.0.0 LTS
- PostgreSQL >= 14.0
- Redis >= 6.2
- Docker >= 24.0
- Kubernetes >= 1.28 (production)
```

### **Development Setup**

```bash
# Clone repository
git clone https://github.com/enterprise/ai-job-chommie.git
cd ai-job-chommie

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Configure with your credentials

# Database setup
cd backend
npx prisma migrate deploy
npx prisma db seed

# Launch development environment
npm run dev
```

### **Production Deployment**

```bash
# Build production images
docker build -t ai-job-chommie:latest .

# Deploy to Kubernetes
kubectl apply -f k8s/

# Verify deployment
kubectl get pods -n ai-job-chommie
```
